# Arbaz URL Shortener

A modern, dark-themed, responsive URL Shortener built with **React**. This app allows users to shorten long URLs, copy them easily, and keep a history of all shortened URLs in the browser.

## Features

- Shorten URLs using TinyURL API
- Copy short URLs to clipboard with a popup notification
- Track URL history with date and original/short URL
- Clear history anytime
- Fully responsive & mobile-friendly
- Dark, modern UI with smooth animations
